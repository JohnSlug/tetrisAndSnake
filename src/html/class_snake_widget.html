Detailed Description

Виджет для отображения игры "Змейка" в Qt. Отрисовывает поле, обрабатывает ввод и обновляет состояние.

  • 'P': пауза/продолжение.

Обновляет игровое состояние (движение, коллизии).

Обновляет игровое состояние:

  • Перемещает змейку.
  • Проверяет съедение яблока и коллизии.
  • Обновляет уровень и скорость.
  • Сохраняет рекорд при завершении игры.
