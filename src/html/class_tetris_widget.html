Detailed Description

Виджет для отображения игры "Тетрис" в Qt. Управляет логикой, отрисовкой фигур и интерфейсом.

Конструктор виджета Tetris.

Инициализирует таймер, структуру tetris_t и запускает игру. Связывает сигналы таймера с обновлением состояния.
